### 미니콘다 설치

- https://www.anaconda.com/download
- 설치 완료 후 anaconda prompt 열어서, $ conda init bash 를 실행!

### 가상환경 생성 

- git bash 를 새로 연다
- 가상환경 생성 : 가상환경이름은 dt 로 생성. 파이썬 버전은 python=3.10 -> $conda create -n 가상환경이름 python=3.10
- 라이브러리 설치 : $conda install openssl numpy scipy matplotlib ipython scikit-learn pandas pillow jupyter seaborn
- 가상환경 실행 $conda activate dt
- 주피터노트북 실행 : $jupyter notebook .
- 가상환경 종료 $conda deactivate
